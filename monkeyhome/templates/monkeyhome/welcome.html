Welcome to Monkey Snowfight!

Remember the simple joy of two monkeys throwing snowballs at each other? In the late 2000s, Monkey Snowfight by Niclas Åberg captured hearts worldwide with its delightfully chaotic multiplayer battles.

My Inspiration

This project is my tribute to that beloved game. While the original had chat features that were "quickly removed for obvious reasons," I saw an opportunity to reimagine what a Monkey Snowfight community could look like with proper moderation and design.

What started as nostalgia became a full-featured real-time chat ecosystem. The game recreation is still a work in progress, but the heart of the community— the chat, the connections, the playful spirit—is alive and thriving.

What's Here Now

Fully Working Chat System - Real-time messaging with rooms, file sharing, and user profiles. Join the public chat, create a group or even direct message a friend.
Game in Development - The snowball fights will return, better than ever
Modern Technology - Built with Django, WebSockets, and responsive design
Community First - Safe, moderated spaces for monkey mayhem
